DocFerry 0.0.10

This release restores the stable DocFerry Cloud endpoint after the Bondie subdomain cutover was found to be unhealthy during smoke testing.

**Cloud endpoint:**

- Uses `https://docferry.fuyonder.tech` as the default Cloud API base URL again.
- Removes the unhealthy `https://docferry.bondie.io` default from the bundled plugin.

**Compatibility:**

- Minimum Obsidian version remains `1.13.0`.

**Release assets:**

- `main.js`
- `manifest.json`
- `styles.css`
