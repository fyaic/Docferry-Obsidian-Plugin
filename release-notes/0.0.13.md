DocFerry 0.0.13

This release addresses the remaining automated review suggestion. There are no user-facing behavior changes.

**Build dependency cleanup:**

- Replaces the `builtin-modules` npm package with Node's built-in `module.builtinModules` in `esbuild.config.mjs`.
- Removes `builtin-modules` from both the plugin and repository-root `package.json` / lockfiles.
- This resolves the review recommendation that `builtin-modules` be replaced with a native alternative.

**No behavior changes:**

- The bundled plugin behavior is unchanged; commands, settings, and publish/import/share flows are identical.

**Compatibility:**

- Minimum Obsidian version remains `1.13.0`.

**Release assets:**

- `main.js`
- `manifest.json`
- `styles.css`
