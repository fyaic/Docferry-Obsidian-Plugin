DocFerry 0.0.16

This release is the Obsidian-review hardening pass for the June 30 public free launch plugin.

**Review hardening:**

- Removes non-plugin server, CLI, importer, and legacy release-note files from the public plugin repository.
- Keeps the public repository focused on the Obsidian plugin source, manifest, privacy notice, build metadata, and release assets.
- Replaces raw heading elements with CSS-classed elements.
- Replaces inline hidden-render styles with a CSS class.
- Restores root ESLint/type-resolution metadata for Obsidian automated review.

**Public billing:**

- Checkout and public paid plans remain disabled in this release.
- Approved access upgrades are handled through request review.

**Release assets:**

- `main.js`
- `manifest.json`
- `styles.css`
- `docferry-obsidian-plugin-0.0.16.zip`
