DocFerry 0.0.17

This release addresses Obsidian Community review feedback and refreshes the local plugin UI.

**Review hardening:**

- Adds the MIT license and aligns package metadata with it.
- Removes `Obsidian` from the plugin description.
- Removes unsupported zip asset copy from current release documentation.
- Declares the bundled cloud upload SDK at the repository root so automated review can resolve its types.
- Avoids Promise-returning click callbacks in UI event handlers.
- Avoids internal calls to the deprecated settings `display()` method.
- Removes CSS `!important` overrides.
- Documents clipboard behavior and upload SDK local/session storage behavior in `PRIVACY.md`.

**UI refresh:**

- Centers the main import view with a larger transparent DocFerry logo.
- Removes the account shortcut from the main view.
- Renames the main shortcut from Config to Settings.
- Keeps share/account sub-pages centered to avoid layout jumps.
- Cleans the settings import panel spacing.
- Refreshes the share completion modal.

**Release assets:**

- `main.js`
- `manifest.json`
- `styles.css`
