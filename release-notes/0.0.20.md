DocFerry 0.0.20

This patch release freezes the current public free plugin package after the July 2 release-line cleanup.

**Release-line cleanup:**

- Reissues the current public free plugin package as `0.0.20` so product, QA, and seed-user testing do not depend on the overloaded `0.0.19` history.
- Includes the product-manager README logo update that uses the PNG DocFerry mark in the public repository docs.
- Records that the old DocFerry hosted service now provides public Privacy Policy and Terms of Service pages.

**Current public free behavior:**

- Public billing controls remain disabled in the plugin UI.
- Connected accounts use the hosted free quota and can request more free quota through the beta-list flow.
- The plugin keeps the upload disclosure before publishing vault content.
- The plugin package remains the Obsidian release artifact set: `main.js`, `manifest.json`, and `styles.css`.

**Release assets:**

- `main.js`
- `manifest.json`
- `styles.css`
